# Taşıburada MVP - Development Plan

## Core Files to Create/Modify:

1. **src/pages/Index.tsx** - Main landing page with user/carrier registration options
2. **src/components/UserRegistration.tsx** - Customer registration form
3. **src/components/CarrierRegistration.tsx** - Carrier registration with vehicle info
4. **src/components/ShipmentRequest.tsx** - Shipment request form with route selection
5. **src/components/CarrierProfile.tsx** - Carrier profile display
6. **src/components/PriceCalculator.tsx** - Automatic price calculation
7. **src/components/RatingSystem.tsx** - Rating and review component
8. **src/lib/types.ts** - TypeScript interfaces for the platform

## Features Implementation:
- User & Carrier registration system
- Route-based matching with map integration
- Vehicle type & capacity management
- Load type selection
- Simple price calculation
- Offer system
- Carrier profiles
- Rating & review system

## Tech Stack:
- React + TypeScript
- Shadcn/UI components
- Tailwind CSS
- Local storage for MVP data persistence
