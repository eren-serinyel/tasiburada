# Project Summary
Taşıburada is a digital transportation marketplace platform that facilitates communication between carriers and customers in a secure, fast, and transparent manner. It allows customers to find suitable carriers with a single click, compare prices, and select additional services like insurance. For carriers, it provides opportunities to gain new customers, optimize routes, and increase income. The platform also offers an admin panel for managing the entire system, reporting, and preventing fraud.

# Project Module Description
The project is structured into several main modules:
- **User Registration**: Allows customers and carriers to register.
- **Carrier Profiles**: Displays carrier information including vehicle details and service areas.
- **Shipment Request System**: Enables customers to create shipment requests with route and load type selections.
- **Smart Pricing System**: Calculates prices based on distance, weight, and vehicle type.
- **Rating and Review System**: Allows users to evaluate carriers based on their experiences.

# Usage
To set up the project, follow these steps:
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Lint the code:
   ```bash
   pnpm run lint
   ```
4. Run the application (local server setup).
